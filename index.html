ระบบที่พัฒนา

การฝึกประสบการณ์วิชาชีพ ณ แผนก Recruitment บริษัท บิ๊กซี ซูเปอร์เซ็นเตอร์ จำกัด (มหาชน) ทำให้ได้เรียนรู้วัฒนธรรมองค์กรที่ส่งเสริมให้พนักงานทุกคนพัฒนาทักษะอยู่ตลอดเวลา มีการจัดประชุมทีมทุกสัปดาห์ การทำ Coaching และ One-on-One กับหัวหน้าทีม รวมถึงการทำ Report สรุปผลงานทุกวันศุกร์

นอกจากเรื่องงานแล้ว สังคมการทำงานที่นี่ยังคอยสนับสนุนกันและกัน มีกิจกรรมร่วมกันเสมอ เช่น การเซอร์ไพรส์วันเกิดคนในทีม ทำให้เกิดการทำงานเป็นทีมที่เข้มแข็ง และได้พัฒนาทักษะรอบด้าน ทั้งการสื่อสาร การนำเสนองาน และการจัดการข้อมูล

ทักษะด้าน HR Recruitment

ได้เรียนรู้การทำงานด้าน HR จากหัวหน้าทีม (คุณบิวตี้) โดยตรง ทำให้ได้รับทักษะ HRM และการวิเคราะห์ข้อมูลของ Recruiter ในทีม เพื่อจัดทำ KPI และประเมิน Performance ตลอดจนกระบวนการสรรหาบุคลากรที่ใช้งานจริงของบิ๊กซี

ทักษะด้าน Data Analysis

ได้เรียนรู้และพัฒนาทักษะด้าน Data Analysis ทั้งจากการศึกษาด้วยตนเอง การได้รับคำแนะนำจากคุณบิวตี้ และการสนับสนุนจากพี่ๆ ในทีมแผนก Data

ทักษะการพัฒนาระบบ (System Development)

นำความรู้พื้นฐานจากห้องเรียนมาใช้จริงและต่อยอดในการพัฒนาระบบ โดยได้รับการสนับสนุนและคำแนะนำจากพี่ๆ ในทีมแผนก IT และ Data

02 Responsibilities

บทบาทหน้าที่
ความรับผิดชอบ

ตลอดระยะเวลา 4 เดือน ได้รับความไว้วางใจให้ปฏิบัติงานจริงใน 3 บทบาทหลัก เพื่อสนับสนุนทีม Recruitment อย่างเต็มรูปแบบ

01
System Developer Intern
รับผิดชอบการออกแบบและพัฒนาระบบกรอกข้อมูลผู้สมัคร System V2 เพื่อใช้งานจริงในแผนก ลดขั้นตอนการทำงานซ้ำซ้อนและจัดเก็บข้อมูลอย่างเป็นระบบ
02
Data Analysis
วิเคราะห์ Data จากการกรอกข้อมูลที่ HR บันทึกลงในระบบ เพื่อนำมาจัดทำ Report Dashboard สำหรับรายงาน Performance ให้กับหัวหน้าทีม HUB (คุณบิวตี้) ใช้ประกอบการตัดสินใจ
03
HR Recruiter
ปฏิบัติงานสรรหาเสมือนพนักงานจริง ทำการพรีสกรีนผู้สมัครในตำแหน่งที่ได้รับมอบหมาย นัดหมายสัมภาษณ์ และติดตามผู้สมัครไปจนสิ้นสุดกระบวนการสรรหา (เซ็นสัญญา)
03 Main Project

Internship 2026 System

ระบบบริหารจัดการข้อมูล
ฝ่าย Recruitment

กระบวนการทั้งหมดถูกพัฒนาขึ้นเพื่อแก้ปัญหาการจัดเก็บข้อมูลแบบกระจาย (Data Silos) เพิ่มความถูกต้อง ความรวดเร็วในการประมวลผลข้อมูล และสนับสนุนการตัดสินใจของผู้บริหารผ่าน Dashboard ได้อย่างมีประสิทธิภาพ โดยแบ่งการใช้งานเป็น 2 ส่วนหลัก:

ฝั่งผู้ใช้งาน (HR Recruiter)

  • บันทึกข้อมูลผู้สมัคร / รายงานผลการโทรและนัดรับ
  • หน้ามอนิเตอร์กระบวนการสรรหาและสัมภาษณ์
  • หน้าแดชบอร์ดสรุปผลการทำงานส่วนตัว
  • ระบบแจ้งเตือนพนักงานลาออก
  • คู่มือการใช้งาน และตั้งค่าปรับแต่งโปรไฟล์ส่วนตัว

ฝั่งผู้ดูแลระบบ (Admin / HRM)

  • Overview Dashboard: ดูยอดโทร/รับนัด (รายวัน/สัปดาห์), จัดอันดับ Performance การทำงาน (ยอดโทร/จ้าง), แจ้งเตือนอัปเดต และป้ายประกาศ
  • Monitor Analytics: วิเคราะห์ข้อมูลทีมแบบเจาะลึก คลิกที่กราฟเพื่อแสดง Pop-up รายชื่อ (CV, Intv, Pass, Sign Contract, WOB, OB) ทันที
  • Recruitment Metrics: ติดตามค่า STD, Active, Vacant, %Fulfillment และความเร็วในการสรรหา
  • Data Filtering: กรองข้อมูลรายวัน/สัปดาห์/เดือน, แยกตามตำแหน่งงานที่ทีมสรรหาอยู่ และแยกตามทีม Leader
  • Data Import: ฟังก์ชันอัปโหลดไฟล์เพื่ออัปเดตค่า STD/Active/Vacant ให้ถูกต้องตรงกับข้อมูลของ HRBP
